# 安装

```npm install runserve -g```


# 运行服务
把当前文件夹作为静态资源服务器根目录
```runserve```

# 自定义配置运行服务

指定根目录/XXX，如下指定/usr为根目录


```runseve -d /usr```


指定服务运行端口


```runseve -p 8080```


指定host


```runseve -h localhost```
